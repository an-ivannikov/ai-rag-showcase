# MythLang Errors

Common runtime errors in MythLang:

- `ManaOverflowError` — occurs when too much mana is channeled.
- `CreatureNotFoundError` — raised if you attempt to summon an unknown creature.
- `ForbiddenRitualError` — thrown when trying to invoke blacklisted spells.
- `EnchantmentFailure` — when the enchantment process is incomplete.
